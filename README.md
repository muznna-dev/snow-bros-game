# Snow Bros Remake

A C++ and SFML remake of the classic Snow Bros arcade game, built as an OOP course project.

## About
This project recreates the classic Snow Bros arcade game using Object-Oriented Programming principles. The game features 10 levels where the player defeats enemies using snowball mechanics and progresses through increasingly difficult stages.

Developed by Muznna Majid as part of a semester OOP project at FAST University.

## Features
- 10 progressively difficult levels
- Single-player and multiplayer modes
- 5 unique enemy types with distinct AI behavior
- Snowball throwing and freezing mechanics
- Power-up system
- User authentication (login/signup)
- Leaderboard and scoring system
- In-game shop
- Persistent storage using file handling

## OOP Concepts Applied
- Encapsulation
- Inheritance (Enemy base class with derived enemy types)
- Polymorphism (virtual update() and draw() methods)
- Abstraction (abstract base classes for enemies, power-ups, UI screens)
- Factory Design Pattern for enemy creation

## Technologies Used
- C++
- SFML (Simple and Fast Multimedia Library)

## How to Run
1. Install SFML on your system
2. Clone the repository
3. Compile and link with SFML
4. Run the executable
